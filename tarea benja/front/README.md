# Vue 3 + Vuetify 3 Project

Este es un proyecto en blanco de Vue 3 con Vuetify 3.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build para producción

```bash
npm run build
```

## Preview de producción

```bash
npm run preview
```

## Tecnologías

- Vue 3
- Vuetify 3
- Vite
- Material Design Icons
